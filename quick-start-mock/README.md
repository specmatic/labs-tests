# quick-start-mock

Automated harness for the upstream `quick-start-mock` lab.

Run from the repo root:

```bash
python3 quick-start-mock/run.py
```

Refresh only the lab report from captured artifacts:

```bash
python3 quick-start-mock/run.py --refresh-report
```

Generated outputs:

- `output/report.json`
- `output/report.html`
