# async-event-flow

Automated harness for the upstream `async-event-flow` lab.

Run from the repo root:

```bash
python3 async-event-flow/run.py
```

Refresh only the lab report from captured artifacts:

```bash
python3 async-event-flow/run.py --refresh-report
```

Generated outputs:

- `output/report.json`
- `output/report.html`
