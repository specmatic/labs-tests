# kafka-sqs-retry-dlq

Automated harness for the upstream `kafka-sqs-retry-dlq` lab.

Run from the repo root:

```bash
python3 kafka-sqs-retry-dlq/run.py
```

Refresh only the lab report from captured artifacts:

```bash
python3 kafka-sqs-retry-dlq/run.py --refresh-report
```

Generated outputs:

- `output/report.json`
- `output/report.html`
