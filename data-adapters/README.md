# data-adapters

Automated harness for the upstream `data-adapters` lab.

Run from the repo root:

```bash
python3 data-adapters/run.py
```

Refresh only the lab report from captured artifacts:

```bash
python3 data-adapters/run.py --refresh-report
```

Generated outputs:

- `output/report.json`
- `output/report.html`
