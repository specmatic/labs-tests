# kafka-avro

Automated harness for the upstream `kafka-avro` lab.

Run from the repo root:

```bash
python3 kafka-avro/run.py
```

Refresh only the lab report from captured artifacts:

```bash
python3 kafka-avro/run.py --refresh-report
```

Generated outputs:

- `output/report.json`
- `output/report.html`
