# order-bff

Automated harness for the upstream `order-bff` lab.

Run from the repo root:

```bash
python3 order-bff/run.py
```

Refresh only the lab report from captured artifacts:

```bash
python3 order-bff/run.py --refresh-report
```

Generated outputs:

- `output/report.json`
- `output/report.html`
